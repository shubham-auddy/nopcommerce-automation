# 🧪 Selenium Automation Framework – NopCommerce Demo

This repository contains a **Selenium + Java Test Automation Framework** for [nopCommerce Demo](https://demo.nopcommerce.com).  
It follows **Page Object Model (POM)** design, **TestNG** for test orchestration, and is built with **Java 21** and **Maven**.  
The project is maintained by **Shubham** and **Ritwik** as a professional, collaborative QA practice project.

---

## 🛠️ Tech Stack

- **Language:** Java 21  
- **Build Tool:** Maven  
- **Automation:** Selenium WebDriver  
- **Test Runner:** TestNG  
- **Dependency Management:** WebDriverManager  
- **Reporting:** Extent Reports / Allure (to be added)  
- **Version Control:** Git (GitHub)  
- **CI/CD:** Jenkins (to be configured)
